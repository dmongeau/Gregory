Class Gregory

Main class for Gregory framework. The class should be instantiated and can be used statically to retrieve current instance from anywhere in your code.

Minimal example

  1.  require Gregory.php;
  2.  
  3.  $config = array(
  4.      'path' => array(
  5.          'pages' => dirname(__FILE__).'/pages'
  6.      )
  7.  );
  8.  $Gregory = new Gregory($config);
  9.  
  10.  // Gregory will look for these php files in $config['path']['pages']
  11.  $Gregory->addRoutes(array(
  12.     '/' => 'home.php',
  13.     '/about.html' => 'about.php'
  14.  ));
  15.  
  16.  $Gregory->boostrap();
  17.  $Gregory->run();
  18.  $Gregory->render();
